My custom arch linux configuration. Started on mylinuxforwork dotfiles, however diverged heavily with my own code to the point thats it essentially all my own.

Includes configurations for:
- niri (window manager)
- waybar (top bar)
- nvim (I used lazyvim plugin)
- tmux (sessions)
- alacritty (terminal)
- fastfetch (pretty)
- walker (app launcher)
- matugen (auto colors)
